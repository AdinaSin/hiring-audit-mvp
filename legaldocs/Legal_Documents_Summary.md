## 1. Privacy Policy

### Key Sections:
- Data Controller identification
- Categories of data collected (Clients, Respondents, Automatic)
- Legal basis for processing (GDPR Article 6)
- Data sharing and disclosure rules
- International transfer safeguards (SCCs, adequacy decisions)
- Data retention periods
- User rights (access, rectification, erasure, portability, objection)
- Security measures
- Cookie usage reference
- Contact information

### GDPR Compliance Features:
✅ Clear identification of data controller  
✅ Lawful basis for each processing activity  
✅ Transparent data subject rights  
✅ International transfer mechanisms  
✅ Defined retention periods  
✅ Security measures documented  
✅ DPO contact information  

---

## 2. Terms of Service

### Key Sections:
- Service description and tiers (L1/L2/L3)
- What the service is NOT (legal advice, guarantee)
- Client obligations (accurate info, designated respondents)
- Payment terms and refund policy
- Intellectual property rights
- Confidentiality obligations
- Disclaimers and warranties
- Limitation of liability
- Indemnification
- Term and termination
- Governing law and dispute resolution
- General provisions

### Critical Clauses:
- **Refund Policy**: Full before initiation, 50% during, none after completion
- **Liability Cap**: 12 months of fees paid
- **Dispute Resolution**: Binding arbitration
- **IP Protection**: Methodology and algorithms protected

## 3. Data Processing Agreement (DPA)

### Key Sections:
- Parties and roles (Controller/Processor)
- Subject matter and processing details
- Categories of data subjects and data types
- Processor obligations (Article 28 GDPR)
- Security measures (technical/organizational)
- Sub-processor list and change notification
- International transfer safeguards
- Data breach notification (48 hours)
- Data subject rights assistance
- Audit rights
- Termination and data return

## 4. Cookie Policy

### Cookie Categories:
| Type | Purpose | Consent Required |
|------|---------|------------------|
| Essential | Core functionality | No |
| Functional | User preferences | Yes |
| Analytics | Usage tracking | Yes |

### Cookies Used:
- `session_id` - Session management (Essential)
- `csrf_token` - Security (Essential)
- `audit_progress` - Progress tracking (Functional)
- `_ga`, `_gid` - Google Analytics (Analytics)
- `cookie_consent` - Preferences storage (Essential)

### User Controls:
- Cookie consent banner on first visit
- "Cookie Settings" link in footer
- Browser settings instructions
- DNT signal disclosure

---

## 5. Service Disclaimer

### Key Disclaimers:
1. **Not Professional Advice** - Not legal, financial, or HR consulting
2. **Self-Reported Data** - Results depend on input quality
3. **Point-in-Time** - Snapshot assessment only
4. **No Guarantees** - No promise of outcomes
5. **Not Individual Evaluation** - Assesses systems, not people

### Liability Limitations:
- No indirect/consequential damages
- No guarantee of issue identification
- No guarantee recommendations will work
- Client responsible for implementation decisions

### Important Notes:
- Results should not be used for employment decisions
- Professional consultation recommended before implementation
- Audit is informational/educational only
