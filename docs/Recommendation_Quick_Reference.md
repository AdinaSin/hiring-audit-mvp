### Block 1: Ownerless Hiring
**Symptom:** No clear executive owner for hiring
**Quick Fix:** Designate CEO/COO as interim owner for 90 days
**Structural Fix:** Create RACI matrix + monthly Governance Forum

### Block 1: Political Prioritization  
**Symptom:** "Who shouts louder" decides priorities
**Quick Fix:** 48-hour cooling period for urgent requests
**Structural Fix:** Weekly prioritization forum with criteria

### Block 2: SLA Theatre
**Symptom:** SLAs exist on paper but not enforced
**Quick Fix:** Audit actual vs. claimed performance (30 days)
**Structural Fix:** SLA dashboard visible to all stakeholders

### Block 2: Capacity Blindness
**Symptom:** Recruiter workload not tracked
**Quick Fix:** Count active roles per recruiter today, set 8-12 max
**Structural Fix:** Build capacity model by role complexity

### Block 3: Interview Bottleneck
**Symptom:** Interviews delayed due to unavailability
**Quick Fix:** Block 4 interview slots/week for key interviewers
**Structural Fix:** Define interviewer pool with capacity SLAs

### Block 3: Feedback Latency
**Symptom:** Interview feedback delayed or missing
**Quick Fix:** Set 24-hour SLA + automated reminders
**Structural Fix:** Escalate delays to manager after 48h

### Block 4: Financial Opacity
**Symptom:** No formal TA budget or cost visibility
**Quick Fix:** List all TA cost components
**Structural Fix:** Build budget template linked to hiring plan

### Block 5: Evaluation Collapse
**Symptom:** No standardized evaluation criteria
**Quick Fix:** Create 5-criteria scorecard template
**Structural Fix:** Interviewer certification program

### Block 6: Key-Person Failure
**Symptom:** Work stops when key recruiter absent
**Quick Fix:** Assign backup for every critical pipeline
**Structural Fix:** Implement buddy system

### Block 7: Shadow AI
**Symptom:** AI used without inventory or governance
**Quick Fix:** Survey team: "What AI tools do you use?"
**Structural Fix:** AI tool approval process

## IMPLEMENTATION TIMELINE

### Week 1-2: Quick Wins
- Designate owners
- Set SLAs (3-5 max)
- Create visibility (even manual reports)
- Stop obvious waste

### Month 1: Foundation
- Document processes (as-is)
- Build capacity model
- Establish cadences
- Define data standards

### Month 2-3: Structure
- Implement dashboards
- Create escalation paths
- Train interviewers
- Consolidate systems

### Quarter 2: Strategic
- Link to business OKRs
- Calculate unit economics
- Validate evaluation quality
- Scale what works
